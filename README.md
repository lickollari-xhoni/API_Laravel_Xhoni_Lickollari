# 🎌 AnimeApp - Esploratore di Anime con Jikan API

AnimeApp è una web application moderna sviluppata con **Laravel 12** che permette agli utenti di sfogliare un vasto database di anime. 
Il progetto utilizza le **API di Jikan** (MyAnimeList) per fornire dati in tempo reale su generi, titoli e dettagli tecnici.

---

## 🚀 Caratteristiche principali

* **Lista Generi**: Visualizzazione dinamica di tutte le categorie anime.
* **Filtro per Categoria**: Visualizzazione degli anime associati a uno specifico genere.
* **Scheda Dettaglio**: Informazioni approfondite (sinossi, voto, episodi) con layout ottimizzato.
* **Design Responsive**: Interfaccia pulita realizzata con **Bootstrap 5**, ottimizzata per mobile e desktop.
* **Navbar Sticky**: Navigazione facilitata con ricerca integrata.

---

## 🛠️ Tech Stack

* **Framework**: [Laravel 11](https://laravel.com)
* **Linguaggio**: PHP 8.2+
* **Frontend**: Blade Templating & Bootstrap 5
* **Build Tool**: Vite
* **API**: [Jikan API](https://jikan.moe/)

## 📂 Struttura del Progetto (Punti Chiave)

1. **app/Http/Controllers**: Gestione della logica e delle chiamate alle API Jikan tramite il client HTTP di Laravel.

2. **resources/views/components/layout.blade.php**: Componente principale che include Navbar e Footer comuni.

3. **routes/web.php**: Definizione delle rotte per generi, liste e dettagli.


## 📸 Anteprima UI
